# Paul Banks

across UK government, industry, and evenings at home. Mostly on systems
where the documentation disappeared sometime around 1997.

Currently working on sensor-driven imaging and signal-processing at
Sens-Tech.

  TMS320 C24x DSP family.

- **[N-Break](https://github.com/banksy-git/nbreak-econet)** —
  an ESP32 bridge between 1980s Econet and modern IP.
  [Write-up.](https://paulbanks.org/projects/econet)
